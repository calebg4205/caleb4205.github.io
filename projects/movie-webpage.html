Example Project

Context

our assignment was to make a webpage about a fictional movie. The reason for doing this was to combine all the basic HTML and CSS we have learned. The goal was to test our knowledge and to prove that we understand the content being taught.

Overview

Me and my group decided to do a project about a man named Steve. He was a normal guy who was tired of always being rejected and left by his girlfriends. The website thoroughly expressed his story and used CSS and HTML to make it presentable.

Takeaways

The most difficult thing about this project was to deal with all the aspects of the webpage. We have never worked with so much code before, especially in different languages. It was hard to remember what syntax went where and how it should be used.
